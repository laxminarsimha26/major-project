# AthiyaHomes 🏨
A full-stack hotel booking application built with the **MERN stack**, designed to provide secure, scalable, and user-friendly booking solutions.

## 🚀 Features
- Hotel search, booking, and payment workflows
- Secure authentication with Passport.js
- Dynamic pricing and real-time availability tracking
- Real-time hotel mapping using MapTiler SDK
- File and media handling with Multer and Cloudinary
- Responsive front-end with React.js and EJS templating

## 🛠️ Tech Stack
- **Frontend**: React.js, EJS, HTML, CSS
- **Backend**: Node.js, Express.js, Mongoose
- **Database**: MongoDB
- **Authentication**: Passport.js
- **Utilities**: Multer, Cloudinary, MapTiler SDK
- **Version Control**: Git, GitHub
- **Deployment**: Render
